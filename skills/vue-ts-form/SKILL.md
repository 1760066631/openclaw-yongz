# Vue TS Form Workflow

## 默认约束
- 表单模型与校验规则保持类型一致。
- 新增与编辑共享逻辑优先抽象复用。
- 默认中文回复，优先修复根因。

## 执行步骤
1. 先定义表单模型类型与初始值。
2. 将校验规则与字段配置集中管理。
3. 明确联动逻辑触发条件，避免隐式副作用。
4. 提交前做类型安全转换，提交后统一反馈结果。
5. 提供重置与回填策略，兼容新增/编辑两种场景。

## 质量检查
- 校验时机一致（输入中、失焦、提交前）。
- 异步校验有防抖或竞态处理。
- 错误提示位置清晰，不遮挡主要操作。
