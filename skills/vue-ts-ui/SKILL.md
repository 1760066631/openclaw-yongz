# Vue TS UI Workflow

## 默认约束
- 优先使用现有组件库能力，不重复造轮子。
- 样式遵循项目既有设计规范与变量体系。
- 默认中文回复，优先最小改动。

## 执行步骤
1. 先识别页面信息层级与主次操作。
2. 选用组件库中的稳定组件组合实现交互。
3. 将业务规则与视觉样式分离，避免逻辑散落模板。
4. 对表格、弹窗、表单等高频结构抽取复用片段。
5. 输出时说明交互规则、状态切换与可扩展点。

## 质量检查
- 桌面端与移动端关键布局不破版。
- 空态、加载态、错误态完整。
- 自定义样式不破坏组件库默认可访问性。
